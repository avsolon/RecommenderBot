# 🎯 Telegram-бот "Мой Рекомендатор" (Recommendation Bot)

Telegram-бот для хранения и управления личными рекомендациями  
(фильмы, книги, еда, товары и др., категории можно дорабатывать)

---

## 🚀 Возможности

- ➕ Добавление рекомендаций  
- 🔍 Поиск по категории и ключевым словам  
- 🎲 Случайная рекомендация  
- 📋 Просмотр всех записей  
- ✏️ Редактирование записей  
- 🗑 Удаление с подтверждением  
- ❓ Встроенная помощь  

---

## 🧠 Как это работает

Каждый пользователь видит только свои записи(возможно доработать версию с общими рекомендациями).  
Данные хранятся в SQLite базе и фильтруются по `user_id`.

### ⚙️ Установка

1. Клонировать репозиторий  
git clone https://github.com/...

из requirements.txt  
3. Указать токен

### 📌 Примеры команд

- /add     — добавить рекомендацию
- /find    — найти
- /random  — случайная рекомендация
- /list    — список
- /edit    — редактировать
- /del     — удалить
- /help    — помощь

---

### 🔒 Безопасность

Все операции выполняются с учётом user_id  
Пользователь не может:  
видеть чужие записи  
редактировать чужие записи  
удалять чужие записи
